# Scarecrow - Yet Another Presentation with Clojure

## Usage

You may have to ensure [Leiningen](http://github.com/technomancy/leiningen) is installed before.

    $ git clone git://github.com/fukamachi/scarecrow
    $ cd scarecrow
    $ lein deps
    $ lein run presen
